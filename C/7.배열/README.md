# 배열
- `동일한 자료형`을 갖는 자료들의 리스트
- 배열의 각 요소는 하나의 변수로 취급
- 배열은 `배열명`과 `첨자`로 구분

## 형태
### 1차원 배열
- 배열의 `첨자가 하나`만 있는 배열
- `첨자의 개수`는 배열 전체의 `구성 요소의 개수`를 의미
- 배열의 각 요소는 배열명과 첨자로 구분

``` C
// 1차원 배열 선언
int a[10] // 자료형 배열명[개수]
```

> ### 1차원 배열 초기화 방법
> 1. 배열 선언 후 초기값 할당
> 2. 배열 선언과 동시에 초기값 할당
> 3. 외부로부터 자료를 입력 받아 초기화

``` C
// 1차원 배열의 사용예시
#include <stdio.h>
void main() {
    static int x[]={1,2,3,4};
    static int y[]={10,20,30,40};
    int i, z[4];
    for (i=0;i<4;++i)
        z[i] = x[i] + y[3-i];
    printf("반대 위치의 배열 요소의 합\n");
    for (i=0;i<4;++i)
        printf("%d + %d = %d\n", x[i], y[3-i], z[i]);
}
```

### 다차원 배열
``` C
// 2차원 배열 선언
int a[3][4] // 자료형 배열명[행의수][열의수]
```

``` C
// 3차원 배열 선언
int a[2][3][4] // 자료형 배열명[면의수][행의수][열의수]
```

### char형 배열
``` C
// char형 배열 선언 
char str[12] // char 배열명[문자열 길이+1]
```

``` C
#include <stdio.h>
void main(){

char name[] = "Hong Gil Dong"; // 문자열 단위로 초기화
char adrs1[6] = {'S','E','O','U','L','\0'};  // 문자 단위로 초기화 "\0"문자를 삽입해야함
char adrs2[6] = {'S','E','O','U','L'};
printf("\n name: %s", name);
printf("\n adrs1: %s", adrs1);
printf("\n adrs2: %s", adrs2); // 의도하지 않는 값이 붙을 수 있음
}
```
