# 파일
## 파일의 구성
파일은 연속된 바이트의 형태로 기록된다.  
파일의 끝은(EoF(End of File))이라 칭한다.

## 파일의 종류
데이터가 저장되는 방식에 따라 구분한다.
1. 텍스트 파일
2. 바이너리 파일

## 파일 객체 생성
``` python
파일객체_참조변수 = open(파일이름, 모드)
```
| 모드 | 설명                               |
| ---- | ---------------------------------- |
| r    | 읽기용도                           |
| w    | 새로운 파일을 쓰는 용도            |
| a    | 파일의 끝에 데이터를 덧붙이는 용도 |

파일을 생성한 이후에는 꼭 작업을 한 후 close()를 통해 닫아주어야 한다.

## 파일 읽기
- `read()` : 특정 개수의 문자를 반환
- `readline()` : 한 라인의 문자열을 반환
- `readlines():list` : 전체 라인의 문자열을 리스트로 반환

## 파일 쓰기
w 모드로 이미 존재하는 파일을 오픈하면 데이터를 삭제한다.
- `write(s:str)`: 파일에 문자열을 작성
